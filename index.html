Catch prompt regressions before you ship.

A hosted CI eval harness for LLM apps. Open a PR, RegressGuard runs your eval suite against the changed prompt or model, and posts a green or red comment before merge.

PR-native eval runs

GitHub App drops in on any repo. Every PR that touches a prompt or model config gets a full eval pass, no manual triggers.

Rich assertion library

Exact-match, contains, regex, JSON-schema, and LLM-as-judge assertions. Mix them per test case. Fail loud, pass fast.

Hard-case dataset

Every regression you catch gets added to your team's dataset. Your evals get sharper with every PR. Portable, exportable, yours.
